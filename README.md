# CSLU(Computer Science Level Up)

본 저장소는 CSLU의 애플리케이션을 담당하는 저장소입니다.

Kotiln 기반의 안드로이드 애플리케이션으로 백엔드와 상호작용하며, 사용자 친화적인 UI를 통해 컴퓨터공학 전공교육을 지원합니다.

## 기술 스택

- **Language**: Kotiln
- **Build Tool**: Android 스튜디오 Dolphin | 2021.3.1 | Kotiln

<br>

## 개발 기간
- **전체 개발기간** : 2024.02.13 ~ 2024.09.26
    - 2024.02 : 프로젝트 기획, 데이터 구조 설정 및 DB 설계
    - 2024.03 ~ 2024.08 : UI 개발 및 서버 연계
    - 2024.09 ~ 2024.12 : 알파테스트 진행 및 설문조사를 통한 기능 개선

<br>

## 주요 기능
- **문제 풀이 기능**
    - 컴퓨터공학 전공 과목에 따른 문제 풀이 및 해설

- **성취도 평가 기능**
    - 사용자가 푼 문제를 기반으로 개인의 성취도를 평가
    - 평가된 성취도를 기반으로 타 사용자와 비교 및 부족한 부분을 시각화
